# Handbook

Welcome to the handbook! This documentation serves as a comprehensive guide to our brand, design system, and content standards.

## How to Use These Docs

This handbook is organized into several sections to help you navigate:

- **Brand Foundation** - Core brand principles and guidelines
- **Writing Style** - Voice, tone, and writing conventions
- **Design System** - Visual design standards and components
- **Components** - Reusable UI components and patterns
- **Content Architecture** - Information structure and organization
- **Case Study Framework** - Guidelines for creating compelling case studies
- **Content Strategy** - Overall content planning and approach
- **Imagery** - Photography, illustration, and visual asset guidelines
- **Technical** - Technical documentation and best practices
- **Bio Content** - Guidelines for writing bios and biographical content

Start with Brand Foundation to understand our core values, then explore other sections as needed for your specific work.
